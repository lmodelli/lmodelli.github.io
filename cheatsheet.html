Tag <Table> <thead> <tbody> <tr> <td> <tfoot>
Name Table Table Head Table body Table row Table Data Table Foot
Description The wrapper element for all HTML tables. The set of rows defining the column headers in a table. The set of rows containing actual table data. The table row container. The table row container. The set of rows defining the footer in the table.

Attribute colspan rowspan
Name Column Span Row Span
Description Defines how many columns a td element should span. Defines how many rows a td element should span.
